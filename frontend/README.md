# HR Portal React Frontend

This directory contains a React-based rewrite of the static HR portal. It uses [Vite](https://vitejs.dev/) for fast development and [React Router](https://reactrouter.com/) for client-side routing.

## Getting Started

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The build output will be generated in the `dist` folder.
